    • еҚұйҷ©жқғйҷҗ

    • зү№зӮ№

      жҷ®йҖҡжқғйҷҗзӣҙжҺҘеңЁAndroidManifest.xmlдёӯеЈ°жҳҺеҚіеҸҜ

      еҚұйҷ©жқғйҷҗйңҖиҰҒиҝӣиЎҢиҝҗиЎҢж—¶жқғйҷҗеӨ„зҗҶ

      еҺҹеҲҷдёҠз”ЁжҲ·дёҖж—ҰеҗҢж„ҸжҹҗдёҖдёӘжқғйҷҗз”іиҜ·пјҢеҗҢз»„зҡ„е…¶е®ғжқғйҷҗдјҡиҮӘеҠЁжҺҲжқғпјҢдҪҶдёҚиҰҒеҹәдәҺжӯӨи§„еҲҷжқҘе®һзҺ°д»»дҪ•йҖ»иҫ‘еҠҹиғҪ

    иҝҗиЎҢж—¶з”іиҜ·жқғйҷҗ

    1.еңЁAndroidManifest.xmlдёӯеЈ°жҳҺжқғйҷҗ

    2.жЈҖжҹҘжқғйҷҗжҳҜеҗҰжҺҲжқғ

    3.и°ғз”ЁonRequestPermissionsResult()ж–№жі•з”іиҜ·жҺҲжқғ

    4.жҠҠжқғйҷҗжҺҲжқғеҗҺжү§иЎҢзҡ„йҖ»иҫ‘е°ҒиЈ…

    ContentProviderеҹәжң¬з”Ёжі•

    1.иҺ·еҸ– ContentResolver е®һдҫӢ

    еңЁ Android дёӯпјҢйҖҡеёёйҖҡиҝҮдёҠдёӢж–ҮпјҲContextпјүжқҘиҺ·еҸ– ContentResolver е®һдҫӢпјҡ

    2.ж·»еҠ ж•°жҚ®

    insert() ж–№жі•з”ЁдәҺеңЁ ContentProvider дёӯжҸ’е…Ҙж–°ж•°жҚ®пјҢеҸӮж•°иҜҙжҳҺеҰӮдёӢпјҡ

    • uriпјҡзӣ®ж Үж•°жҚ®зҡ„ URI

    • ContentValuesпјҡеӯҳеӮЁй”®еҖјеҜ№ж•°жҚ®

    3.еҲ йҷӨж•°жҚ®

    delete() ж–№жі•з”ЁдәҺеҲ йҷӨ ContentProvider дёӯзҡ„ж•°жҚ®пјҢеҸӮж•°иҜҙжҳҺеҰӮдёӢпјҡ

    • uriпјҡзӣ®ж Үж•°жҚ®зҡ„ URI

    • selectionпјҡеҲ йҷӨжқЎд»¶

    • selectionArgsпјҡжқЎд»¶еҸӮж•°ж•°з»„

    4.жҹҘиҜўж•°жҚ®

    query() ж–№жі•з”ЁдәҺд»Һ ContentProvider жҹҘиҜўж•°жҚ®пјҢеҸӮж•°иҜҙжҳҺеҰӮдёӢпјҡ

    • uriпјҡйңҖиҰҒи®ҝй—®зҡ„ж•°жҚ®зҡ„ URIпјҢдҫӢеҰӮпјҡcontent://com.example.app.provider/city

    • projectionпјҡйңҖиҰҒжҹҘиҜўзҡ„еҲ—ж•°з»„пјҢеҰӮжһңдј е…Ҙ nullпјҢеҲҷиҝ”еӣһжүҖжңүеҲ—

    • selectionпјҡиҝҮж»ӨжқЎд»¶пјҲWHERE еӯҗеҸҘпјүпјҢеҸҜдҪҝз”Ё ? еҚ дҪҚз¬Ұ

    • selectionArgsпјҡиҝҮж»ӨжқЎд»¶еҜ№еә”зҡ„еҸӮж•°ж•°з»„

    • sortOrderпјҡжҺ’еәҸи§„еҲҷ

    5.жӣҙж–°ж•°жҚ®

    update() ж–№жі•з”ЁдәҺжӣҙж–° ContentProvider дёӯзҡ„ж•°жҚ®пјҢеҸӮж•°иҜҙжҳҺеҰӮдёӢпјҡ

    • uriпјҡзӣ®ж Үж•°жҚ®зҡ„ URI

    • ContentValuesпјҡеӯҳеӮЁйңҖиҰҒжӣҙж–°зҡ„еӯ—ж®өе’ҢеҖј

    • selectionпјҡжҢҮе®ҡжӣҙж–°жқЎд»¶

    • selectionArgsпјҡжқЎд»¶еҸӮж•°ж•°з»„

    еҲӣе»әContentProvider

    • ж–°е»әдёҖдёӘзұ»з»§жүҝContentProviderе№¶йҮҚеҶҷ6дёӘж–№жі•(onCreate(),query(),insert(),update(),delete(),getType())

    • е®һзҺ°ж•°жҚ®ж“ҚдҪң

      • query()пјҡж №жҚ®дј е…Ҙзҡ„ URI жҹҘиҜўж•°жҚ®пјҢе№¶иҝ”еӣһдёҖдёӘ Cursor еҜ№иұЎгҖӮ
      • insert()пјҡж №жҚ®дј е…Ҙзҡ„ URI жҸ’е…Ҙж•°жҚ®пјҢиҝ”еӣһж–°жҸ’е…Ҙж•°жҚ®зҡ„ URIгҖӮ
      • update()пјҡж №жҚ® URI жӣҙж–°ж•°жҚ®пјҢиҝ”еӣһеҸ—еҪұе“Қзҡ„иЎҢж•°гҖӮ
      • delete()пјҡж №жҚ® URI еҲ йҷӨж•°жҚ®пјҢиҝ”еӣһеҲ йҷӨзҡ„иЎҢж•°гҖӮ
      • getType()пјҡиҝ”еӣһжҢҮе®ҡ URI еҜ№еә”зҡ„ж•°жҚ® MIME зұ»еһӢгҖӮ

    • й…ҚзҪ® URI еҢ№й…ҚеҷЁ дҪҝз”Ё UriMatcher еҜ№дј е…Ҙзҡ„ URI иҝӣиЎҢеҢ№й…ҚпјҢд»ҘдҫҝзЎ®е®ҡиҜ·жұӮзұ»еһӢе№¶жү§иЎҢзӣёеә”ж“ҚдҪңгҖӮеҸҜд»ҘеңЁйқҷжҖҒд»Јз Ғеқ—дёӯж·»еҠ еҢ№й…Қи§„еҲҷ

    • еңЁ AndroidManifest.xml дёӯжіЁеҶҢ ContentProvider еңЁжё…еҚ•ж–Үд»¶дёӯжіЁеҶҢдҪ зҡ„ ContentProviderпјҡ
