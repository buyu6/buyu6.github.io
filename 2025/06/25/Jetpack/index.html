    Jetpackе…Ёе®¶зҰҸ

    ViewModel

    дҪңз”Ё

    её®еҠ©ActivityеҲҶжӢ…дёҖйғЁеҲҶе·ҘдҪңпјҢдё“й—Ёз”ЁдәҺеӯҳж”ҫдёҺз•Ңйқўзӣёе…ізҡ„ж•°жҚ®зҡ„пјҢд»–еҸҜд»ҘдҝқиҜҒеңЁжүӢжңәеұҸе№•еҸ‘з”ҹж—ӢиҪ¬ж—¶дёҚдјҡиў«йҮҚж–°еҲӣе»әпјҢж•°жҚ®дёҚдјҡдёўеӨұ

    з”ҹе‘Ҫе‘Ёжңҹ

    з”Ёжі•

    • ж·»еҠ дҫқиө–

    • еҲӣе»әдёҖдёӘViewModelзұ»з»§жүҝиҮӘViewModel()

    • жҠҠз•Ңйқўзӣёе…ізҡ„ж•°жҚ®йғҪж”ҫе…Ҙе…¶дёӯ

    • еңЁActivityжҲ–fragmentдёӯеҲӣе»әе…¶е®һдҫӢ

    • жһ„е»әзӣёе…ійҖ»иҫ‘

    еғҸViewModelдј йҖ’еҸӮж•°

    • ж·»еҠ еҸӮж•°

    • ж–°е»әViewModelFactoryзұ»пјҢе®һзҺ°ViewModelProvider.FactoryжҺҘеҸЈ

    • еҲ©з”ЁSharedPreferencesдҝқеӯҳж•°жҚ®

    • иҜ»еҸ–ж•°жҚ®пјҢе°Ҷж•°жҚ®дј е…ҘеҲ°ViewModelFactoryзҡ„жһ„йҖ еҮҪж•°дёӯ

    Lifecycles

    дҪңз”Ё

    д»–еҸҜд»Ҙи®©д»»дҪ•дёҖдёӘзұ»иҪ»жқҫең°ж„ҹзҹҘеҲ°Activityз”ҹе‘Ҫе‘Ёжңҹ

    з”Ёжі•

    • ж–°е»әдёҖдёӘзұ»

    • еҖҹеҠ©жіЁи§ЈиғҪеҠӣзј–еҶҷйҖ»иҫ‘д»Јз Ғ

    • иҺ·еҸ–LifecycleOwerе®һдҫӢпјҢActivityжҲ–иҖ…Fragmentжң¬иә«е°ұжҳҜдёҖдёӘLifecycleOwerе®һдҫӢ

    • еңЁжҙ»еҠЁдёӯиҝӣиЎҢи°ғз”Ё(д№ҹеҸҜд»ҘеңЁд»»дҪ•ең°ж–№и°ғз”Ёlifecycle.currentStateжқҘдё»еҠЁиҺ·еҸ–еҪ“еүҚз”ҹе‘Ҫе‘ЁжңҹзҠ¶жҖҒ)

    LiveData

    еҹәжң¬з”Ёжі•

    • ж·»еҠ дҫқиө–

    • еҲӣе»әLiveData

      • MutableLiveData: еҸҜеҸҳзҡ„ LiveData зұ»еһӢпјҢе…Ғи®ёж•°жҚ®жӣҙж–°гҖӮ
      • LiveData: дёҚеҸҜеҸҳзҡ„ LiveData зұ»еһӢпјҢеҸӘиғҪи§ӮеҜҹж•°жҚ®пјҢдёҚиғҪзӣҙжҺҘдҝ®ж”№ж•°жҚ®гҖӮ

    • еңЁ Activity жҲ– Fragment дёӯи§ӮеҜҹ LiveData

      еңЁ Activity жҲ– Fragment дёӯпјҢдҪ еҸҜд»ҘйҖҡиҝҮ observe() ж–№жі•и§ӮеҜҹ LiveDataгҖӮеҪ“ж•°жҚ®еҸ‘з”ҹеҸҳеҢ–ж—¶пјҢLiveData жүҚдјҡеҸ‘йҖҒжӣҙж–°гҖӮ

    mapе’ҢswitchMap

    • map

      • дҪңз”Ёпјҡе°Ҷе®һйҷ…еҢ…еҗ«ж•°жҚ®зҡ„LiveDataе’Ңд»…з”ЁдәҺи§ӮеҜҹж•°жҚ®зҡ„LiveDataиҝӣиЎҢиҪ¬жҚў

      • з”Ёжі•пјҡ

    • switchMap

      • дҪҝз”ЁеңәжҷҜпјҡеҪ“ViewModelдёӯжҹҗдёӘliveDataеҜ№иұЎжҳҜи°ғз”ЁеҸҰеӨ–зҡ„ж–№жі•иҺ·еҸ–зҡ„ж—¶еҖҷз”ЁswitchMap

      • з”Ёжі•пјҡ

        • searchQuery жҳҜдёҖдёӘеҢ…еҗ«жҗңзҙўе…ій”®еӯ—зҡ„ LiveDataгҖӮ
        • switchMap е°ҶжҜҸж¬Ўжӣҙж–°зҡ„ searchQuery жҳ е°„еҲ°дёҖдёӘж–°зҡ„ LiveDataпјҲиҝҷйҮҢжЁЎжӢҹдёәжҗңзҙўз»“жһңпјүгҖӮ
        • еҪ“ searchQuery жӣҙж–°ж—¶пјҢswitchMap дјҡеҸ–ж¶Ҳд№ӢеүҚзҡ„жҹҘиҜўе№¶и§ҰеҸ‘ж–°зҡ„жҹҘиҜўгҖӮ

    Room

    еҹәжң¬зү№еҫҒ

    Room жҳҜ Jetpack жҸҗдҫӣзҡ„дёҖдёӘж•°жҚ®еә“еә“пјҢз®ҖеҢ–дәҶ SQLite зҡ„ж“ҚдҪңгҖӮе®ғжҸҗдҫӣдәҶдёҖдёӘжҠҪиұЎеұӮпјҢе…Ғи®ёејҖеҸ‘иҖ…жӣҙж–№дҫҝгҖҒжӣҙе®үе…Ёең°и®ҝй—®жң¬ең°ж•°жҚ®еә“гҖӮRoom жҸҗдҫӣдәҶжіЁи§ЈпјҲannotationпјүж”ҜжҢҒпјҢз»“еҗҲ Kotlin зҡ„ж•°жҚ®зұ»е’Ңжһ¶жһ„з»„д»¶пјҢиғҪеӨҹжӣҙй«ҳж•Ҳең°з®ЎзҗҶж•°жҚ®жҢҒд№…еҢ–ж“ҚдҪңгҖӮ

    Room жҸҗдҫӣдәҶдёҖдёӘ DAOпјҲж•°жҚ®и®ҝй—®еҜ№иұЎпјү еұӮжқҘиҝӣиЎҢж•°жҚ®еә“ж“ҚдҪңпјҢDAO дҪҝз”Ё SQL жҹҘиҜўиҜӯеҸҘжқҘдёҺж•°жҚ®еә“дәӨдә’гҖӮRoom дјҡеңЁзј–иҜ‘ж—¶з”ҹжҲҗе®һйҷ…зҡ„е®һзҺ°д»Јз ҒпјҢеҮҸе°‘дәҶжүӢеҠЁзј–еҶҷ SQL иҜӯеҸҘзҡ„з№ҒзҗҗгҖӮ

    • з®ҖеҚ•жҳ“з”ЁпјҡзӣёжҜ”зӣҙжҺҘдҪҝз”Ё SQLiteпјҢRoom жҸҗдҫӣдәҶжӣҙз®ҖеҚ•гҖҒзұ»еһӢе®үе…Ёзҡ„ APIгҖӮ
    • SQLite е®Ңе…Ёж”ҜжҢҒпјҡе®ғжҳҜеҹәдәҺ SQLite зҡ„пјҢеҸҜд»Ҙжү§иЎҢ SQL жҹҘиҜўгҖҒжҸ’е…ҘгҖҒжӣҙж–°е’ҢеҲ йҷӨж“ҚдҪңгҖӮ
    • жіЁи§Јж”ҜжҢҒпјҡдҪҝз”ЁжіЁи§Је®ҡд№үж•°жҚ®еә“е®һдҪ“гҖҒDAO е’Ңж•°жҚ®еә“зүҲжң¬гҖӮ
    • дёҺ LiveData е…је®№пјҡRoom ж”ҜжҢҒдёҺ LiveData з»“еҗҲдҪҝз”ЁпјҢеҪ“ж•°жҚ®ж”№еҸҳж—¶пјҢRoom еҸҜд»ҘиҮӘеҠЁйҖҡзҹҘи§ӮеҜҹиҖ…гҖӮ

    ж·»еҠ дҫқиө–е’ҢжҸ’д»¶

    е®ҡд№үе®һдҪ“зұ»(Entity)

    • @Entity жіЁи§Јж ҮиҜҶиҝҷдёӘзұ»жҳҜдёҖдёӘж•°жҚ®еә“иЎЁгҖӮ
    • @PrimaryKey жіЁи§ЈжҢҮе®ҡдё»й”®пјҢе№¶еҸҜд»Ҙи®ҫзҪ® autoGenerate = true и®© Room иҮӘеҠЁз”ҹжҲҗдё»й”®гҖӮ

    е®ҡд№үDao

    • Roomе…ій”®йғЁеҲҶпјҢжүҖжңүж•°жҚ®еә“ж“ҚдҪңеқҮе°ҒиЈ…еңЁиҝҷйҮҢ
    • @Dao жіЁи§Јж ҮиҜҶиҝҷдёӘжҺҘеҸЈжҳҜ DAOгҖӮ
    • @Insert жіЁи§Јз”ЁдәҺжҸ’е…Ҙж•°жҚ®гҖӮ
    • @Query жіЁи§Је®ҡд№ү SQL жҹҘиҜўиҜӯеҸҘгҖӮ
    • @UpdateжіЁи§Јз”ЁдәҺжӣҙж–°ж•°жҚ®
    • @DeleteжіЁи§Јз”ЁдәҺеҲ йҷӨж•°жҚ®
    • еҲ©з”Ёйқһе®һдҪ“зұ»еҸӮж•°жқҘеўһеҲ ж”№ж•°жҚ®ж—¶йғҪиҰҒз”Ё**@Query**жіЁи§Ј

    е®ҡд№үж•°жҚ®еә“(Database)

    • @Database жіЁи§Јз”ЁдәҺе®ҡд№үж•°жҚ®еә“пјҢе№¶жҢҮе®ҡеҢ…еҗ«зҡ„е®һдҪ“зұ»е’Ңж•°жҚ®еә“зүҲжң¬пјҢе®һдҪ“зұ»д№Ӣй—ҙз”ЁйҖ—еҸ·йҡ”ејҖгҖӮ
    • abstract fun userDao() ж–№жі•жҸҗдҫӣеҜ№ DAO зҡ„и®ҝй—®гҖӮ
    • йҖҡиҝҮ Room.databaseBuilder() ж–№жі•иҺ·еҸ–ж•°жҚ®еә“е®һдҫӢ
    • з”ұдәҺж•°жҚ®еә“ж“ҚдҪңеқҮдёәиҖ—ж—¶ж“ҚдҪңж•…йңҖиҰҒеңЁеӯҗзәҝзЁӢдёӯиҝӣиЎҢ

    ж•°жҚ®еә“еҚҮзә§

    1. жӣҙж–°зүҲжң¬е’ҢеҢ…еҗ«зҡ„е®һдҪ“зұ»
    2. е®һзҺ° Migrationзҡ„еҢҝеҗҚзұ»
    3. жһ„е»әе®һдҫӢж—¶еҠ е…ҘaddMigrationsж–№жі•

    WorkManager

    дҪңз”Ё

    еӨ„зҗҶдёҖдәӣиҰҒжұӮе®ҡж—¶жү§иЎҢзҡ„д»»еҠЎпјҢеҸҜд»Ҙж №жҚ®ж“ҚдҪңзі»з»ҹзҡ„зүҲжң¬иҮӘеҠЁйҖүжӢ©жҳҜдҪҝз”ЁAlarmManagerиҝҳжҳҜJobSchedulerе®һзҺ°гҖӮеҸҰеӨ–иҝҳиғҪж”ҜжҢҒе‘ЁжңҹжҖ§д»»еҠЎпјҢй“ҫејҸд»»еҠЎеӨ„зҗҶзӯүеҠҹиғҪгҖӮе®ғеҫҲйҖӮеҗҲжү§иЎҢдёҖдәӣе®ҡжңҹе’ҢжңҚеҠЎеҷЁдәӨдә’зҡ„д»»еҠЎпјҢжҜ”еҰӮе‘ЁжңҹжҖ§зҡ„еҗҢжӯҘж•°жҚ®

    еҹәжң¬з”Ёжі•

    1. ж·»еҠ дҫқиө–

    1. е®ҡд№үеҗҺеҸ°д»»еҠЎе№¶е®һзҺ°е…·дҪ“зҡ„д»»еҠЎйҖ»иҫ‘

      • иҝ”еӣһResult.success()иЎЁзӨәжҲҗеҠҹ
      • иҝ”еӣһResult.failure()иЎЁзӨәеӨұиҙҘ
      • иҝ”еӣһResult.retry()д№ҹиЎЁзӨәеӨұиҙҘпјҢеҸӘжҳҜеҸҜд»Ҙз»“еҗҲWorkRequest.Builderзҡ„setBackoffCriteria()ж–№жі•жқҘйҮҚж–°жү§иЎҢд»»еҠЎ

    2. й…ҚзҪ®иҜҘеҗҺеҸ°д»»еҠЎзҡ„иҝҗиЎҢжқЎд»¶е’ҢзәҰжқҹдҝЎжҒҜпјҢе№¶жһ„е»әеҗҺеҸ°д»»еҠЎиҜ·жұӮ

      • OneTimeWorkRequest жқҘжү§иЎҢдёҖж¬ЎжҖ§д»»еҠЎ
      • PeriodicWorkRequest жқҘжү§иЎҢе®ҡжңҹд»»еҠЎгҖӮ

    3. е°ҶиҜҘеҗҺеҸ°д»»еҠЎиҜ·жұӮдј е…ҘWorkManagerзҡ„enqueue()зҡ„ж–№жі•дёӯпјҢзі»з»ҹдјҡеңЁеҗҲйҖӮзҡ„ж—¶й—ҙиҝҗиЎҢ

    еӨ„зҗҶеӨҚжқӮд»»еҠЎ

    • и®ҫзҪ®зәҰжқҹдҝЎжҒҜ

    • еҸ–ж¶ҲеҗҺеҸ°д»»еҠЎиҜ·жұӮ
