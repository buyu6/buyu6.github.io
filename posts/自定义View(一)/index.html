  • format取值类型：

    格式类型 含义说明
    reference 资源引用，例如 @string/app_name、@drawable/icon
    string 字符串文本，例如 "Hello"
    integer 整数，例如 42
    boolean 布尔值，例如 true、false
    color 颜色值，例如 #FF0000 或 @color/primary
    dimension 尺寸值，例如 16dp、12sp
    float 浮点数，例如 3.14
    enum 枚举值（需配合 <enum> 使用）
    flag 位标志（可多选，需配合 <flag> 使用）
